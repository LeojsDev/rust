# Exercícios de Rust

Este diretório contém exercícios e experimentos em Rust.

Objetivo:
- Praticar conceitos básicos e intermediários de Rust.
- Registrar exemplos de código e anotações sobre a linguagem.
- Servir como referência para estudos futuros.

Estrutura:
- Cada exercício pode estar em um projeto separado ou em arquivos individuais.
- Use `cargo` para compilar e executar os exemplos.

Este repositório é voltado para aprendizado e prática de Rust.
