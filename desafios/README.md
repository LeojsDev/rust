# Desafios Rust

Coleção de desafios e exercícios práticos para aprender e dominar a linguagem Rust.

## Sobre

Este repositório contém uma série de desafios progressivos para desenvolver habilidades em Rust, desde conceitos básicos até tópicos mais avançados.

## Estrutura

- **Nível Iniciante**: Conceitos fundamentais da linguagem
  - Variáveis e tipos de dados
  - Funções e controle de fluxo
  - Ownership e borrowing

- **Nível Intermediário**: Conceitos mais avançados
  - Pattern matching
  - Traits e generics
  - Tratamento de erros

- **Nível Avançado**: Tópicos complexos
  - Lifetimes
  - Async/await
  - Macros

## Como Usar

1. Clone este repositório
2. Navegue até o desafio desejado
3. Leia as instruções no arquivo específico
4. Implemente a solução
5. Execute os testes com `cargo test`

## Requisitos

- Rust 1.56 ou superior
- Cargo

## Instalação do Rust

Para instalar o Rust, visite [rustup.rs](https://rustup.rs/)
